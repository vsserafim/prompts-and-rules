# Prompts para estilos de escrita

## Extração de estilo de escrita

```
Analise o seguinte texto e identifique suas principais características de escrita.
Descreva o tom (formal, informal, técnico, persuasivo, etc.), o estilo (objetivo, descritivo, narrativo, etc.), a complexidade da linguagem, a estrutura das frases o uso de figuras de linguagem e qualquer outro aspecto relevante. 
Depois,com base nessa análise gere um trecho de prompt que possa ser inserido em outros prompts para instruir o modelo a escrever ou melhorar textos mantendo essas mesmas características.
```
